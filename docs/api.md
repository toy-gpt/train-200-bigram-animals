# API Reference

This repository uses the `toy_gpt_train` package for core functionality.

For API documentation, see the reference implementation:

- [toy_gpt_train API Reference](https://toy-gpt.github.io/train-200-bigram/api/)

## This Repository

This repository provides:

- **Corpus**: `corpus/001_animals.txt` - structured animal sentences
- **Wrappers**: Thin modules that configure paths and run demos
- **Artifacts**: Pretrained model outputs in `artifacts/`

No new public API is exported beyond re-exports from `toy_gpt_train`.
